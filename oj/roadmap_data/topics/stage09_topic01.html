🎯 Mục tiêu bài học

  • Hiểu các khái niệm cơ bản: đỉnh, cạnh, đồ thị có hướng/vô hướng.
  • Biết hai cách lưu trữ: ma trận kề và danh sách kề.

📘 Ví dụ

Cho đồ thị vô hướng có 4 đỉnh, 4 cạnh: 1–2, 1–3, 2–4, 3–4.

💡 Danh sách kề

1: 2 3
2: 1 4
3: 1 4
4: 2 3

💻 C++

#include <bits/stdc++.h>
using namespace std;
int main(){
    int n,m;cin>>n>>m;
    vector<vector<int>> adj(n+1);
    for(int i=0;i<m;i++){
        int u,v;cin>>u>>v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    for(int i=1;i<=n;i++){
        cout<<"Đỉnh "<<i<<": ";
        for(int v:adj[i]) cout<<v<<" ";
        cout<<"\\n";
    }
}

💻 Python

n,m=map(int,input().split())
adj=[[] for _ in range(n+1)]
for _ in range(m):
    u,v=map(int,input().split())
    adj[u].append(v)
    adj[v].append(u)
for i in range(1,n+1):
    print("Đỉnh",i,":",*adj[i])

🧩 Bài tập luyện tập

  1. Chuyển đổi từ ma trận kề sang danh sách kề.
  2. Đếm bậc (degree) của từng đỉnh.
  3. Xác định xem đồ thị có cạnh song song không.
💡 Gợi ý lời giải 1: Đếm bậc 
for i in range(1,n+1):
    print("Degree(",i,") =",len(adj[i]))
