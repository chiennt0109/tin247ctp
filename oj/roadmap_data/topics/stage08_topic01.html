🎯 Mục tiêu bài học

  • Phân biệt rõ ba cấu trúc: Stack (ngăn xếp), Queue (hàng đợi), Deque (hai đầu).
  • Hiểu ứng dụng thực tế của từng loại trong xử lý dữ liệu luồng.

📘 Lý thuyết cơ bản

🔹 Stack (Ngăn xếp): Lấy ra phần tử cuối cùng được thêm vào (“Last In, First Out”).
🔹 Queue (Hàng đợi): Lấy ra phần tử đầu tiên được thêm vào (“First In, First Out”).
🔹 Deque (Double-ended Queue): Có thể thêm/xóa ở cả hai đầu.

💻 Ví dụ minh họa

C++

#include <bits/stdc++.h>
using namespace std;
int main(){
    stack<int> st;
    queue<int> q;
    deque<int> dq;
    for(int i=1;i<=3;i++){
        st.push(i);
        q.push(i);
        dq.push_back(i);
    }
    cout<<"Stack top: "<<st.top()<<"\\n";
    cout<<"Queue front: "<<q.front()<<"\\n";
    cout<<"Deque back: "<<dq.back()<<"\\n";
}

Python

from collections import deque
st = []
q = deque()
dq = deque()

for i in range(1,4):
    st.append(i)
    q.append(i)
    dq.append(i)

print("Stack top:", st[-1])
print("Queue front:", q[0])
print("Deque back:", dq[-1])
