🎯 Mục tiêu bài học

  • Hiểu cách tìm kiếm phần tử trong mảng (Linear Search, Binary Search).
  • Phân biệt khi nào dùng tuyến tính, khi nào nên dùng nhị phân.
  • Thực hành với ví dụ C++ và Python.

📘 Lý thuyết cơ bản

Tìm kiếm tuyến tính (Linear Search): duyệt qua toàn bộ mảng và so sánh từng phần tử.

Tìm kiếm nhị phân (Binary Search): chỉ dùng được khi mảng đã được sắp xếp, chia đôi để thu hẹp phạm vi tìm kiếm nhanh hơn.

Thuật toánĐộ phức tạpMô tả
Linear SearchO(n)Duyệt toàn bộ mảng
Binary SearchO(log n)Chia đôi dần khoảng tìm kiếm

🧠 Thực hành nhanh

C++

#include <bits/stdc++.h>
using namespace std;
int main() {
    int n, x;
    cin >> n >> x;
    vector a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    int pos = -1;
    for (int i = 0; i < n; i++) if (a[i] == x) pos = i;
    cout << (pos != -1 ? "Tim thay" : "Khong tim thay");
}

Python

n, x = map(int, input().split())
a = list(map(int, input().split()))
print("Tim thay" if x in a else "Khong tim thay")

🧩 Bài tập luyện tập

  1. Nhập mảng gồm n phần tử, nhập giá trị x, tìm xem x có trong mảng hay không.
  2. Nhập mảng tăng dần, tìm vị trí phần tử x bằng tìm kiếm nhị phân.
  3. Viết chương trình đếm số lần xuất hiện của x trong mảng.
💡 Xem lời giải mẫu 
# Python: Binary Search
def binary_search(a, x):
    l, r = 0, len(a) - 1
    while l <= r:
        m = (l + r) // 2
        if a[m] == x:
            return m
        elif a[m] < x:
            l = m + 1
        else:
            r = m - 1
    return -1
print(binary_search([1,2,3,4,5,6], 4))
