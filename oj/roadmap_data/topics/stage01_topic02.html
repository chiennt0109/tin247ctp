🎯 Mục tiêu bài học

  • Hiểu được cấu trúc rẽ nhánh: if, else.
  • Biết cách so sánh, kiểm tra điều kiện trong chương trình.

📘 1️⃣ Cấu trúc điều kiện trong C++

if (điều_kiện) {
    // Nếu đúng, thực hiện
} else {
    // Nếu sai, thực hiện khác
}

🐍 2️⃣ Trong Python

if điều_kiện:
    # khi đúng
else:
    # khi sai

🧩 Ví dụ: Kiểm tra số chẵn hay lẻ

C++Python
 
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n; cin >> n;
    if (n % 2 == 0)
        cout << "Even";
    else
        cout << "Odd";
}
 
n = int(input())
if n % 2 == 0:
    print("Even")
else:
    print("Odd")

🎯 Ứng dụng mở rộng

  • Kiểm tra tuổi hợp lệ.
  • Điểm ≥ 5 thì Đậu, ngược lại Rớt.
  • So sánh hai số, in ra số lớn hơn.

💡 Bài tập

  1. Nhập một số nguyên, cho biết là dương, âm hay bằng 0.
  2. Nhập điểm, in ra kết quả: “Đậu” hoặc “Rớt”.
  3. Nhập 3 số, in ra số lớn nhất.
