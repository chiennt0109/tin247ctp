🎯 Mục tiêu bài học

  • Hiểu vì sao cần đọc / ghi file thay vì chỉ nhập tay.
  • Biết cách đọc dữ liệu từ file đầu vào (.INP).
  • Biết cách ghi kết quả ra file đầu ra (.OUT).

1. Tại sao phải dùng file?

Trong các kỳ thi lập trình, dữ liệu đầu vào / đầu ra thường nằm trong file:
BAI1.INP chứa dữ liệu, BAI1.OUT chứa kết quả.

2. Cách đọc / ghi file trong C++

#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream fin("BAI1.INP");   // mở file để đọc
    ofstream fout("BAI1.OUT");  // mở file để ghi

    int a, b;
    fin >> a >> b;              // đọc 2 số
    fout << (a + b);             // ghi kết quả

    return 0;
}

Lưu ý: File .INP và .OUT nằm cùng thư mục chạy chương trình.

3. Cách đọc / ghi file trong Python

fin = open("BAI1.INP", "r")
fout = open("BAI1.OUT", "w")

a, b = map(int, fin.read().split())
fout.write(str(a + b))

fin.close()
fout.close()

4. Bài mẫu kiểu đề thi thật

Đề: SUM.INP chứa hai số a, b. Hãy tạo SUM.OUT ghi tổng a+b.

💻 Lời giải mẫu C++ 
#include <bits/stdc++.h>
using namespace std;
int main() {
    ifstream fin("SUM.INP");
    ofstream fout("SUM.OUT");
    long long a, b;
    fin >> a >> b;
    fout << (a + b);
    return 0;
}
🐍 Lời giải mẫu Python 
fin = open("SUM.INP", "r")
fout = open("SUM.OUT", "w")

nums = fin.read().split()
a = int(nums[0])
b = int(nums[1])
fout.write(str(a + b))

fin.close()
fout.close()

🧩 Bài tập tự luyện

  1. PRODUCT.INP chứa hai số nguyên. Ghi ra PRODUCT.OUT giá trị a*b.
  2. MAXNUM.INP chứa nhiều số, ghi ra MAXNUM.OUT số lớn nhất.
  3. COUNTZERO.INP chứa ma trận R×C, ghi ra COUNTZERO.OUT số phần tử bằng 0.

➡ Đây là bước đầu để làm bài thi tự động chấm bằng file.
