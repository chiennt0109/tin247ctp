🎯 Mục tiêu bài học

  • Hiểu nguyên lý chia để trị.
  • Biết cách tách bài toán lớn thành các phần nhỏ, giải từng phần rồi kết hợp.

📘 Ý tưởng

“Chia để trị” gồm 3 bước:

  1. Chia bài toán thành 2 hoặc nhiều phần nhỏ.
  2. Giải quyết từng phần bằng đệ quy.
  3. Kết hợp các kết quả để ra kết quả cuối.

💻 Ví dụ: Tính tổng dãy số bằng chia để trị

C++

#include <bits/stdc++.h>
using namespace std;
int sumRange(vector<int>& a, int l, int r){
    if(l==r) return a[l];
    int m=(l+r)/2;
    return sumRange(a,l,m)+sumRange(a,m+1,r);
}
int main(){
    int n;cin>>n;
    vector<int> a(n);
    for(int i=0;i<n;i++)cin>>a[i];
    cout<<sumRange(a,0,n-1);
}

Python

def sum_range(a, l, r):
    if l == r:
        return a[l]
    m = (l + r) // 2
    return sum_range(a, l, m) + sum_range(a, m + 1, r)
n = int(input())
a = list(map(int, input().split()))
print(sum_range(a, 0, n-1))

🧩 Bài tập luyện tập

  1. Tính tổng các phần tử trong mảng bằng chia để trị.
  2. Tìm phần tử lớn nhất trong mảng bằng chia để trị.
  3. Đếm số phần tử chẵn trong mảng bằng chia để trị.
💡 Xem lời giải mẫu 
# Python
def max_range(a, l, r):
    if l == r: return a[l]
    m = (l + r)//2
    return max(max_range(a,l,m), max_range(a,m+1,r))
a = [1,7,3,9,2,8]
print(max_range(a,0,len(a)-1))
