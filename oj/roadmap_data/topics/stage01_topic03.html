🎯 Mục tiêu bài học

  • Hiểu khái niệm vòng lặp – lặp lại công việc nhiều lần.
  • Phân biệt giữa for và while.
  • Thực hành tính tổng, đếm, in dãy số.

📘 1️⃣ Vòng lặp for trong C++

for (int i = 1; i <= n; i++) {
    // khối lệnh lặp
}

🐍 2️⃣ Trong Python

for i in range(1, n + 1):
    # khối lệnh lặp

🧩 Ví dụ: Tính tổng từ 1 đến n

C++Python
 
int n, sum = 0;
cin >> n;
for (int i = 1; i <= n; i++)
    sum += i;
cout << sum;
 
n = int(input())
total = 0
for i in range(1, n + 1):
    total += i
print(total)

🔁 3️⃣ Vòng lặp while

C++Python
 
int i = 1;
while (i <= n) {
    cout << i << " ";
    i++;
}
 
i = 1
while i <= n:
    print(i, end=" ")
    i += 1

💡 Bài tập luyện tập

  1. Nhập n, in ra dãy 1 → n.
  2. Tính tổng các số chẵn từ 1 → n.
  3. Tính giai thừa n! (5! = 1×2×3×4×5).

🧠 Thử thách mở rộng

Hãy dùng while thay cho for để làm lại các bài trên.
