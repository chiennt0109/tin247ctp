📘 Giải thích

Một hàm đệ quy có dạng:

// C++
returnType function(parameters) {
    if (điều kiện dừng) return kết_quả;
    return function(tham_số_thay_đổi);
}

Nếu không có điều kiện dừng, chương trình sẽ gọi vô hạn và tràn stack.

def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)
print(factorial(int(input())))

# Python
def fib(n):
    if n <= 1: return n
    return fib(n - 1) + fib(n - 2)
