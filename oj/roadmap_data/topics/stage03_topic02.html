🎯 Mục tiêu bài học

  • Biết cách cài đặt các thuật toán sắp xếp cơ bản.
  • Hiểu vì sao có nhiều cách sắp xếp khác nhau.
  • So sánh hiệu quả giữa Selection Sort, Insertion Sort và Bubble Sort.

📘 Lý thuyết

Mỗi thuật toán sắp xếp có đặc điểm riêng:

TênÝ tưởngĐộ phức tạp
Selection SortChọn phần tử nhỏ nhất đưa lên đầuO(n²)
Insertion SortChèn từng phần tử vào vị trí đúngO(n²)
Bubble SortSo sánh cặp kề, đẩy phần tử lớn ra cuốiO(n²)

💻 Ví dụ thực hành

C++ – Bubble Sort

#include <bits/stdc++.h>
using namespace std;
int main() {
    int n; cin >> n;
    vector a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n - 1; i++)
        for (int j = 0; j < n - i - 1; j++)
            if (a[j] > a[j + 1])
                swap(a[j], a[j + 1]);
    for (int x : a) cout << x << " ";
}

Python – Insertion Sort

def insertion_sort(a):
    for i in range(1, len(a)):
        key = a[i]
        j = i - 1
        while j >= 0 and a[j] > key:
            a[j + 1] = a[j]
            j -= 1
        a[j + 1] = key
a = list(map(int, input().split()))
insertion_sort(a)
print(*a)

# Python - Selection Sort
a = [5, 3, 8, 1, 2]
for i in range(len(a)):
    min_idx = i
    for j in range(i+1, len(a)):
        if a[j] < a[min_idx]:
            min_idx = j
    a[i], a[min_idx] = a[min_idx], a[i]
print(a)
