📘 1️⃣ Cấu trúc cơ bản của chương trình C++

Với C++, em hãy tưởng tượng mỗi khi em muốn xây một ngôi nhà (viết một chương trình), em luôn luôn phải xây một cái móng và bộ khung sườn y hệt nhau. Đây chính là "bộ khung" thần thánh đó. Em phải ghi nhớ nó nhé, gần như 99% các bài tập của em đều bắt đầu bằng cái khung này:

#include <bits/stdc++.h>
using namespace std;

int main() {
    // Code sẽ viết ở đây
    return 0;
}

return 0; là cách báo cho máy tính biết chương trình đã chạy xong và chạy đúng.

  • #include <bits/stdc++.h>: gọi tất cả công cụ cần thiết (thư viện).
  • using namespace std;: giúp viết lệnh ngắn gọn hơn.
  • int main() { ... }: cửa chính để chương trình bắt đầu chạy.
Program └ main() ├ nhập dữ liệu ├ xử lý └ in kết quả
