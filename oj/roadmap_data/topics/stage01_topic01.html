1️⃣ Biến là gì?

Trong lập trình, biến (variable) giống như một chiếc hộp dùng để lưu trữ dữ liệu.

5️⃣ Phạm vi của biến (Scope)

Phạm vi của biến là nơi mà biến đó có thể được sử dụng trong chương trình.

5.1 Biến toàn cục (Global variable)

Khai báo ngoài hàm main().

#include <bits/stdc++.h>
using namespace std;

int x = 10;

int main() {
    cout << x;
}

Biến x có thể sử dụng ở mọi nơi trong chương trình.

5.2 Biến cục bộ (Local variable)

Khai báo bên trong hàm.

int main() {
    int a = 5;
    cout << a;
}

Biến a chỉ tồn tại trong hàm main.

🏫 Loa phát thanh của trường → ai cũng nghe → giống biến toàn cục.
👨‍🏫 Giáo viên nói nhỏ trong lớp → chỉ lớp nghe → giống biến cục bộ.
