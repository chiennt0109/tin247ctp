📘 1️⃣ Cấu trúc cơ bản của chương trình C++

Với C++, em hãy tưởng tượng mỗi khi em muốn xây một ngôi nhà (viết một chương trình), em luôn luôn phải xây một cái móng và bộ khung sườn y hệt nhau. Đây chính là "bộ khung" thần thánh đó. Em phải ghi nhớ nó nhé, gần như 99% các bài tập của em đều bắt đầu bằng cái khung này:

#include <bits/stdc++.h>
using namespace std;

int main() {
    // Code sẽ viết ở đây
    return 0;
}

(Thầy thêm return 0; vào nhé, đây là một phần quan trọng để báo cho máy tính biết chương trình đã chạy xong và chạy đúng.)
Giải thích siêu nhanh (em chưa cần hiểu sâu, chỉ cần biết mặt):

  • #include <bits/stdc++.h>: Giống như em gọi "Tổng đài", em bảo máy tính là: "Tôi cần tất cả các công cụ mà ông có!" (đây gọi là thư viện).
  • using namespace std;: Để lát nữa mình gõ lệnh cho nó ngắn gọn (thay vì gõ std::cout thì chỉ cần gõ cout).
  • int main() { ... }: Đây là "cửa chính" của ngôi nhà. Máy tính luôn chạy vào cái cửa main này đầu tiên để xem em ra lệnh gì.

🧑‍🏫 5️⃣ Thử ngay

🧩 Bài tập luyện tập

  1. Nhập 2 số nguyên, in ra tổng, hiệu, tích, thương.
  2. Nhập tên và tuổi, in ra: Xin chào [tên], bạn [tuổi] tuổi!
  3. Nhập n, tính tổng các số từ 1 đến n.
