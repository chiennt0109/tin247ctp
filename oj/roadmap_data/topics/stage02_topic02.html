🎯 Mục tiêu bài học

  • Hiểu chuỗi là gì (string) và cách máy tính lưu chữ.
  • Nhập chuỗi từ bàn phím (cả có dấu cách và không có dấu cách).
  • Nối chuỗi, đo độ dài chuỗi, truy cập từng ký tự.
  • Làm các tác vụ cơ bản hay gặp: đếm số chữ 'a', đảo ngược chuỗi, so sánh chuỗi.

1. Khai báo chuỗi

C++Python
 
#include <bits/stdc++.h>
using namespace std;
int main() {
    string s = "Xin chao";
    cout << s;
}
 
s = "Xin chao"
print(s)

2. Nhập chuỗi

Trường hợp 1: Nhập một từ (không có dấu cách)

C++Python
 
string name;
cin >> name;
cout << "Hello " << name;
 
name = input()
print("Hello", name)

Trường hợp 2: Nhập cả dòng (có dấu cách, ví dụ: "Nguyen Van A")

C++Python
 
string full_name;
getline(cin, full_name);
cout << "Chao ban " << full_name;

Lưu ý: nếu trước đó bạn đã dùng cin >> ... thì cần thêm cin.ignore(); trước getline để bỏ ký tự xuống dòng còn sót lại.

 
full_name = input()
print("Chao ban", full_name)

3. Độ dài chuỗi và truy cập ký tự

C++Python
 
string s = "abcXYZ";
cout << s.size() << "\\n"; // in ra độ dài
cout << s[0];              // 'a'
 
s = "abcXYZ"
print(len(s))   # độ dài
print(s[0])     # 'a'

4. Ví dụ bài toán hay gặp: Đếm số ký tự 'a'

C++Python
 
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    getline(cin, s);
    int count_a = 0;
    for (int i = 0; i < (int)s.size(); i++) {
        if (s[i] == 'a' || s[i] == 'A') count_a++;
    }
    cout << count_a;
    return 0;
}
 
s = input()
count_a = 0
for ch in s:
    if ch == 'a' or ch == 'A':
        count_a += 1
print(count_a)

5. Bài tập tự luyện

  1. Nhập họ và tên, in ra câu: Xin chào, [Tên]! (chỉ lấy tên cuối).
  2. Nhập một chuỗi, in ra chuỗi đảo ngược.
  3. Cho chuỗi, đếm số lượng chữ số (0-9) xuất hiện trong đó.
🔍 Gợi ý / Lời giải mẫu Python cho bài "đảo ngược chuỗi" 
s = input()
rev = s[::-1]   # cắt ngược chuỗi
print(rev)
🔍 Gợi ý / Lời giải mẫu C++ cho bài đếm chữ số 
#include <bits/stdc++.h>
using namespace std;
int main() {
    string s;
    getline(cin, s);
    int cnt = 0;
    for (char c : s) {
        if (c >= '0' && c <= '9') cnt++;
    }
    cout << cnt;
    return 0;
}
