Stage 01 · Chuỗi ký tự cơ bản

Chuỗi ký tự: từ chữ, tên học sinh đến xử lý dữ liệu

Chuỗi giúp chương trình làm việc với chữ viết, tên người, mã học sinh, câu văn, mật khẩu, số điện thoại và cả những con số đang được lưu dưới dạng ký tự.

Mục tiêu bài học

Sau bài này, học sinh có thể dùng chuỗi để xử lý tên, câu văn, ký tự và dữ liệu số đang nằm trong văn bản.

Hiểu chuỗi là gì và cách máy tính lưu chữ.
Nhập chuỗi từ bàn phím, gồm cả chuỗi có dấu cách và không có dấu cách.
Nối chuỗi, đo độ dài chuỗi, truy cập từng ký tự.
Làm các tác vụ cơ bản: đếm chữ 'a', đảo ngược chuỗi, so sánh chuỗi.
Ví dụ đời sống: Tên học sinh "Nguyen Van An", mã học sinh "HS2026", số điện thoại "0934353569" đều có thể được lưu bằng chuỗi.
1. Khai báo chuỗi

Chuỗi là một dãy ký tự. Trong C++, ta dùng kiểu string; trong Python, chuỗi được viết trực tiếp bằng dấu nháy.

#include <bits/stdc++.h>
using namespace std;

int main() {
    string s = "Xin chao";
    cout << s;
    return 0;
}
s = "Xin chao"
print(s)

Chuỗi dưới dạng các ô ký tự

Chuỗi "Xin chao" có thể hình dung như một dãy ô, mỗi ô chứa một ký tự.

2. Nhập chuỗi

Có hai tình huống quan trọng: nhập một từ và nhập cả dòng có dấu cách.

1
Nhập một từ

Dùng khi dữ liệu không có dấu cách, ví dụ: An, CodePath, ABC123.

2
Nhập cả dòng

Dùng khi dữ liệu có dấu cách, ví dụ: Nguyen Van A, Hoc lap trinh C++.

string name;
cin >> name;
cout << "Hello " << name;
string full_name;
getline(cin, full_name);

Lưu ý: nếu trước đó bạn đã dùng cin >> ... thì cần thêm cin.ignore(); trước getline để bỏ ký tự xuống dòng còn sót lại. 
name = input()
print("Hello", name)

full_name = input()
print("Chao ban", full_name)
3. Độ dài chuỗi và truy cập ký tự

Chuỗi giống một mảng ký tự. Ta có thể lấy độ dài và truy cập từng ký tự bằng chỉ số.

string s = "abcXYZ";
cout << s.size() << "\n";  // in ra độ dài
cout << s[0];               // 'a'
s = "abcXYZ"
print(len(s))   # độ dài
print(s[0])     # 'a'

Thử truy cập ký tự

Chuỗi abcXYZ có độ dài 6. Ký tự tại i = 0 là a.
4. Các bài toán chuỗi hay gặp

Khi mới học, hãy tập nhìn chuỗi như một dãy ký tự rồi duyệt từng ký tự bằng vòng lặp.

Ví dụ: Đếm số ký tự 'a'
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    getline(cin, s);

    int count_a = 0;
    for (int i = 0; i < (int)s.size(); i++) {
        if (s[i] == 'a' || s[i] == 'A') {
            count_a++;
        }
    }

    cout << count_a;
    return 0;
}
s = input()
count_a = 0

for ch in s:
    if ch == 'a' or ch == 'A':
        count_a += 1

print(count_a)

5. Ép kiểu và chuyển đổi số ↔ chữ

Đây là phần rất hay gặp khi làm bài chuỗi: dữ liệu nhìn giống số nhưng thực chất đang là ký tự hoặc chuỗi.

Ký tự chữ số → số

Nếu c = '7', muốn lấy số 7:

int x = c - '0';

Vì các ký tự '0', '1', ..., '9' nằm liên tiếp trong bảng mã.

Số 0..9 → ký tự chữ số

Nếu x = 7, muốn tạo ký tự '7':

char c = x + '0';

Chỉ dùng trực tiếp khi x là một chữ số từ 0 đến 9.

Chuỗi số → số nguyên

Nếu s = "2026", muốn đổi thành số 2026:

int n = stoi(s);

Trong Python dùng int(s).

Số nguyên → chuỗi

Nếu n = 2026, muốn đổi thành chuỗi "2026":

string s = to_string(n);

Trong Python dùng str(n).

Ví dụ: tính tổng các chữ số trong chuỗi
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    getline(cin, s);

    int sum = 0;
    for (char c : s) {
        if (c >= '0' && c <= '9') {
            int digit = c - '0';
            sum += digit;
        }
    }

    cout << sum;
    return 0;
}
s = input()
sum_digits = 0

for ch in s:
    if '0' <= ch <= '9':
        digit = ord(ch) - ord('0')
        sum_digits += digit

print(sum_digits)
Tại sao c - '0' lại ra số?

Trong bảng mã, các ký tự chữ số được xếp liên tiếp:

Ký tự'0''1''2'...'7'
Giá trị tương đối012...7

Vì vậy, khoảng cách từ '0' đến '7' chính là 7.

Thử chuyển đổi

Ký tự '7' đổi thành số 7.
6. Bài tập tự luyện

Các bài đầu ôn lại nội dung gốc; các bài sau luyện ép kiểu và chuyển đổi số - chữ.

Bài tập cơ bản

  1. Nhập họ và tên, in ra câu: Xin chào, [Tên]! trong đó [Tên] là từ cuối cùng.
  2. Nhập một chuỗi, in ra chuỗi đảo ngược.
  3. Cho chuỗi, đếm số lượng chữ số 0..9 xuất hiện trong đó.
  4. Nhập một chuỗi, đếm số chữ cái thường, chữ cái hoa và chữ số.

Bài tập ép kiểu / số ↔ chữ

  1. Nhập chuỗi bất kỳ, tính tổng các chữ số xuất hiện trong chuỗi. Ví dụ "A12B3" có tổng chữ số là 6.
  2. Nhập một ký tự chữ số c, in ra giá trị số tương ứng. Ví dụ '8' in ra 8.
  3. Nhập một số nguyên x từ 0 đến 9, in ra ký tự chữ số tương ứng.
  4. Nhập số nguyên n, đổi sang chuỗi và in ra số lượng chữ số của n.
  5. Mật mã chữ số: nhập chuỗi, mỗi chữ số tăng thêm 1 theo vòng tròn. Ví dụ "A9B2" thành "A0B3".
Gợi ý / Lời giải mẫu Python cho bài đảo ngược chuỗi 
s = input()
rev = s[::-1]   # cắt ngược chuỗi
print(rev)
Gợi ý / Lời giải mẫu C++ cho bài đếm chữ số 
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    getline(cin, s);

    int cnt = 0;
    for (char c : s) {
        if (c >= '0' && c <= '9') {
            cnt++;
        }
    }

    cout << cnt;
    return 0;
}
Gợi ý C++: mật mã chữ số tăng thêm 1 
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    getline(cin, s);

    for (char &c : s) {
        if (c >= '0' && c <= '9') {
            int digit = c - '0';
            digit = (digit + 1) % 10;
            c = char(digit + '0');
        }
    }

    cout << s;
    return 0;
}
